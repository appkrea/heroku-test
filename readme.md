# AddYum Monorepo (clean start)

## Requirements
- Node.js 18+

## Install
```bash
npm install
```

## Dev
```bash
npm run dev
```
- Frontend: http://localhost:5173
- Backend:  http://localhost:3001
- API via frontend proxy: http://localhost:5173/api/health

## Prod (local simulation)
```bash
npm run prod
```
- App served from backend: http://localhost:3001
- API: http://localhost:3001/api/health
